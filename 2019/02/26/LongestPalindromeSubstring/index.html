Longest Palindrome Substring

Given a string s, find the longest palindromic substring in s. You may assume that the maximum length of s is 1000.

解法1:
寻找最长的回文字串，一个比较重要的思想就是它是由首尾相同的字符加上同样是回文的字串组成。subtring[i, j] = c[i] + substring[i - 1, j + 1] + c[j], (c[i] == c[j]). 我们可以认为寻找最长回文字符串就是尽可能在原有能找到的回文串基础上延伸。这样我们就可以得到一个思路就是，遍历string中每一个字符，或者每两个字符，在此基础上向两边不断扩张，需要注意，我们既要寻找奇数长度的回文串，也要寻找偶数长度的回文串。时间复杂度为O(N^2)

解法2:
我们也可以利用动态规划的思想，将之前已经得到的回文串，用于我们接下来的回文串判断。我们想得到字符串中任一子字符串substring[i, j]是否为回文串,需要之前得到的substring[i + 1, j - 1]的信息息。我们可以遍历每个字符j,对于每个字符j，遍历从0到j到字符，设为i, 判断字符j和字符i是否相等，以及substring[i + 1, j - 1]是否为回文字符串。时间复杂度为O(N^2)

解法3:
Manacher算法，O(N)时间复杂度，具体原理请见https://blog.csdn.net/dyx404514/article/details/42061017
